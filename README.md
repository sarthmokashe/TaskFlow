# 🚀 TaskFlow

TaskFlow is a modern task management web application built using PHP, HTML, CSS, JavaScript, and JSON storage.

It helps users manage tasks efficiently with a clean UI, secure authentication system, progress tracking, and responsive dashboard design.

---

## ✨ Features

- User Authentication
- Add / Edit / Delete Tasks
- Task Completion Tracking
- Progress Analytics
- Priority Management
- Responsive Glassmorphism UI
- Animated Dashboard
- Secure Password Hashing

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- PHP
- JSON Database
